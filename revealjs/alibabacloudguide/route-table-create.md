To create a routing table:

-   Go to the VPC console,

-   Click on VPC,

-   Select the region,

-   Click Route Tables,

-   Click Create Route Table,

-   Resource Group: this is the name of the group,

-   VPC: this is the VPC,

-   Name: this is the name of the routing table,

-   Description: this is the description,

-   Click on OK.
